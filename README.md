# Plataforma-multi-franquicia-JS
plataforma multi franquicia es un sistema digital que permite administrar varias franquicias desde un solo entorno centralizado. Facilita la gestión de ventas, inventarios, usuarios, reportes, facturación y control operativo para múltiples sucursales o marcas, manteniendo independencia en la operación de cada una, pero con supervisión global.
